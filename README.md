# car-dealership-capstone
Dealership Review & Management System
This repository contains the Capstone Project for the IBM Full Stack Software Developer Professional Certificate. It is a cloud-native, microservices-based application designed to manage car dealership listings and customer reviews with integrated sentiment analysis.

**Project Details**

Project Name: fullstack_developer_capstone
Repository Name: xrwvm-fullstack_developer_capstone
**Key Features**
User Authentication: Secure Sign-up and Login system built using Django and React.

Dealer Management: Browse dealership listings and filter results by State.

Sentiment Analysis: Integrated Flask microservice that uses Natural Language Processing (NLP) to determine if user reviews are positive, negative, or neutral.

Microservices Architecture: * Django: Main web application and user authentication.

Node.js/Express: Management of dealership and review data.

Python/Flask: Sentiment analysis processing.

CI/CD & Cloud: Automated deployment via GitHub Actions to IBM Cloud Kubernetes Service.

**Tech Stack**
Frontend: React.js, Bootstrap/Tailwind CSS

Backend: Django (Python), Node.js, Express

Database: MongoDB (Reviews), SQLite/PostgreSQL (Users/Cars)

DevOps: Docker, Kubernetes, GitHub Actions

Testing: RESTful API testing via cURL
